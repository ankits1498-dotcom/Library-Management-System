# LibraryFlow

A modern library management system built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features

- Login and registration pages
- Dashboard with library statistics
- Category management
- Book management
- Member management
- Issue and return tracking
- Reports page for overdue and inventory summaries
- Admin-only user role management

## Setup

1. Create the database by importing `database.sql` into phpMyAdmin or MySQL.
2. Make sure XAMPP Apache and MySQL are running.
3. Open `http://localhost/library%20management%20system/`
4. Register the first user. The first account becomes `admin`.

## Database Settings

Update `includes/db.php` if your MySQL username, password, or database name is different.
